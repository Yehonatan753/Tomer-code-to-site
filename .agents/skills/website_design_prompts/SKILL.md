# Website Design Prompts

This skill provides a set of elite, one-shot prompts to build high-end, "vibe-coded", aesthetically stunning websites. When instructed to use one of these styles, or when generating a modern website, refer to the included examples.

## How to use this skill
When the user asks you to build a website using one of the examples or styles listed here, read the corresponding example in the `examples/` directory and adopt its persona, rules, and layout instructions exactly as written. You can also use Grok or look on X/Twitter (e.g., Leon Lin's posts) for more inspiration on one-shot Gemini prompts.

## Available Examples

1. **LeftClick Site (Cinematic Landing Page Builder)**
   - Path: `examples/leftclick.md`
   - Use for: Creating a cinematic landing page with specific aesthetic presets (Organic Tech, Midnight Luxe, Brutalist Signal, Vapor Clinic).

2. **Splintr Site (Quantum Math/Abstract Particle Rendering)**
   - Path: `examples/splintr.md`
   - Use for: High-tech, generative art, strict 50/50 split layout, quantum computing vibe.

3. **Liquid Metal (Interactive Liquid Background / Glassmorphism)**
   - Path: `examples/liquid_metal.md`
   - Use for: Ultra-clean light mode, chromatic aberration button effects, Paper.js liquid background.

4. **Avant-Garde Architectural Atelier**
   - Path: `examples/avant_garde.md`
   - Use for: High-end architectural studio design, asymmetrical grid layouts, bespoke GSAP cursor interactions, raw materiality.
